# React Task Manager

A simple task management application built using React Hooks.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive user interface

## Technologies Used

- React
- JavaScript (ES6)
- CSS3

## Installation

```bash
npm install
npm start
```

## Future Improvements

- Edit task feature
- Task filtering
- Local storage support
